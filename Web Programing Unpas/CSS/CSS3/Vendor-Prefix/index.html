Prefix Vendor (awalan vendor/browser yang bersangkutan) sintaks khusus yang harus ditambahkan untuk beberapa property CSS3. Vendor Prefixes -webkit- (Chrome, Safari, Opera versi terbaru) -moz- (Firefox) -ms- (Internet explorer) -o- (Opera versi terdahulu) cek pada Website shouldiprefix.com, caniuse.com, http://pleeease.iamvdo.me/play/ untuk melihat browser mana yg belum support property CSS3 Prefix Vendor
